Portal Web - Inspiration Church

Diseño y desarrollo integral de la plataforma web oficial de la organización comunitaria "Inspiration Church". Este proyecto incluyó la creación de un sitio web dinámico y responsivo utilizando React para el frontend, Node.js para el backend y MongoDB como base de datos. La plataforma permite a los usuarios acceder a información sobre eventos, servicios, recursos y actividades de la iglesia, así como facilitar la interacción con la comunidad a través de formularios de contacto y suscripción a boletines.

Desafío Técnico: Implementar un sistema robusto de autenticación y autorización de usuarios (manejo de roles), garantizando la seguridad de los datos personales. Además, se integró un módulo de base de datos optimizado para manejar agendas de eventos concurrentes y la carga eficiente de recursos multimedia.

Sistema POS y ERP integral - Medimerkar

Desarrollo de un sistema de Punto de Venta (POS) y automatización de procesos para la cadena de supermercados "Medimerkar". La plataforma centraliza en una sola interfaz web tres pilares críticos del negocio: la contabilidad automatizada (registro de transacciones, caja diaria y reportes financieros), el control de inventarios en tiempo real (alertas de stock bajo, entrada y salida de mercancía) y la gestión de nómina para el personal de las diferentes sucursales.

Desafío Técnico: Diseñar una arquitectura de base de datos altamente relacional y optimizada para procesar un alto volumen de transacciones simultáneas sin perder rendimiento. El mayor reto consistió en la automatización de la lógica contable (procesos ETL internos) para que cada venta afecte inmediatamente los niveles de inventario y los balances financieros de la empresa de manera exacta.

Sistema de gestión de aprendizaje - Educonnect

Desarrollo de una plataforma educativa web diseñada para instituciones medianas. Permite la creación de cursos, carga de material multimedia, entrega de tareas y cuenta con un módulo de comunicación en vivo (chats y foros de discusión) para fomentar la interacción entre estudiantes y profesores.

Desafío Técnico: Garantizar la seguridad y el control de accesos mediante roles de usuario estrictos (Administrador, Profesor, Estudiante) y optimizar la carga de archivos multimedia pesados en el servidor.
